# F1-2026-analisis-real
simulaciones monte carlo y dashboard de F1 2026 

F1 2026 - Análisis de Probabilidades de Campeon
Despues de 3 carreras (Australia, China, Japon), desarrolle un dashboard interactivoque simulo 3000 escenarios para predecir el campeonato de Formula 1 2026

Tecnologias utilizadas:
PYTHON: Simulaciones Monte Carlo, procesamiento de datos, limpieza y transformacio.
POWER BI: Visualización interactiva, modelado de datos, DAX
SQL: Consultas de datos históricos.

archivos:
F1_2026.py - codigo de simulaciones pendiente de subir
archivos csv - datos procesados para powwr bi
iamgen del dashboard - vista previa

Resultados claves:
Mercedes lidera con 99.7% de probabildad de campeonato
Antonelli es el favorito pata el titulo despues de (3 carreras)
Ferrari y Mclaren compiten por el segundo lugar
Mapa interactivo con todos los circuitos de la temporada 2026
